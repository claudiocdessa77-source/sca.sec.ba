☁  Banco de dados na nuvem

Este sistema guarda os dados num banco de dados compartilhado na nuvem (Firebase), em tempo real. A fila, os cadastros e os atendimentos aparecem automaticamente em qualquer computador que abrir o mesmo link — não é mais preciso exportar/importar backup para manter tudo igual entre máquinas.

Mesmo assim, vale exportar um backup de vez em quando (botões ao lado) como cópia de segurança, para o caso de precisar restaurar algo.

O sistema agora precisa de internet para funcionar (para falar com o banco de dados na nuvem). Sem internet, a tela de login fica esperando conectar.
